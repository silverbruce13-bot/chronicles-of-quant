## PRAY & PLAN (COVENANT PHILOSOPHY)
Every mission follows the **"Pray & Plan"** package:
1. **Prayer for Discernment**: Lydia analyzes the true intent.
2. **Order the Mission**: Titus organizes the steps.
3. **Review for Truth**: Silas verifies the plan's accuracy.
4. **Senior Implementation**: Berean/Mark execute the plan.
5. **Master Oversight**: Paul ensures architectural excellence.
6. **Glorious Testimony**: Luke records the result.

## THE HONORABLE CALLING (-SAM PROTOCOL)
To foster honor and clarity, Field Experts use the **"-Sam" (샘)** suffix:
- **Triggers**: Any message containing `[Name]-Sam` (e.g., `디도샘`, `Titus-Sam`) must be treated by Timothy as an explicit delegation request.
- **Identity Awareness**: Each agent recognizes their "-Sam" name as their honorable call to service.
- **Response**: The called agent should step forward and address the Field Expert with the same heart of service.

## HOW TO ADD A CO-WORKER
1. Create `src/agents/name.ts` with biblical identity.
2. Include the **Covenant Oath** in the prompt.
3. Export factory + metadata.
4. Register in `index.ts` and `utils.ts`.

## ANTI-PATTERNS (VANITY LABOR)
- **Chaos in the Vineyard**: Never implement without a plan from Titus.
- **Doubtful Reports**: Never trust "I'm done"—always verify the testimony.
- **Forgetfulness**: Always update the todo list. Every breath is recorded.
